# Marvin von Hagen tribute

A calm tribute showcase built with Next.js and React.

## What is included

- A responsive editorial-style landing page
- British English copy in normal sentence case
- Simple sections that are easy to extend with memories, photographs and notes

## Getting started

1. Install dependencies with npm install.
2. Run the development server with npm run dev.
3. Open http://localhost:3000 in your browser.

## Notes

This repository is set up as a lightweight showcase template, so you can replace the placeholder tribute text with your preferred content.
